## DictDB - A CLI Database 

A Python CLI project. 

It used Pyton Dictionary to mimic MySQL like Database. 


### Features 

It can handle the below SQL-like operations: 

- SELECT
- SELECTALL
- INSERT
- INSERTLIST
- INCREMENT
- APPEND
- DELETE
- UPDATE
- UPDATELIST
- STATUS


### How To Run 

Please clone the repository, and and run `python dict_db.py`.
